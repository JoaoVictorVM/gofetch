# gofetch

A fast, minimal system information tool for the terminal, written in Go — inspired by [neofetch](https://github.com/dylanaraps/neofetch) and [fastfetch](https://github.com/fastfetch-cli/fastfetch).

It displays your machine's specs (OS, CPU, memory, disk, motherboard and more) next to an ASCII art logo of your operating system, styled with colors.

> **Status:** work in progress — v1 targets Windows.

## Usage

```
gofetch
```
